cyber-moz-tools

Ferramentas e guias de cibersegurança em português para Moçambique.

Guias

  • Passwords Seguras — como criar e gerir senhas fortes
  • Phishing e Golpes — como identificar golpes online
  • Wi-Fi Segura — proteger a tua rede em casa
  • Contactos Úteis — reportar crimes digitais em Moçambique
  • Privacidade no WhatsApp — proteger a tua conta

Scripts

  • check-headers.py — verificar cabeçalhos de segurança de sites moçambicanos
  • gerar-senha.py — gerar passwords aleatórias fortes

Como usar

git clone https://github.com/mabolacha58-hub/cyber-moz-tools.git
cd cyber-moz-tools
python scripts/gerar-senha.py 16
python scripts/check-headers.py tvsucesso.co.mz

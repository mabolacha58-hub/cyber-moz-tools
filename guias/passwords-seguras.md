# Guia de Passwords Seguras

Proteger as tuas contas começa com uma boa password.

## O que NÃO usar

- Nome, apelido, data de nascimento
- `123456`, `senha`, `password`
- Nome do teu bairro, cidade ou animal de estimação
- Palavras comuns em português (ex: `chapa`, `machamba`, `biscato`)

## Regras básicas

1. **Mínimo 12 caracteres**
2. Misturar: maiúsculas, minúsculas, números, símbolos
3. **Nunca** repetir a mesma password em sites diferentes
4. Trocar a password se souberes que houve um vazamento

## Como criar uma password forte (fácil de lembrar)

Escolhe uma frase curta em português e transforma:

> "Eu gosto de ver a bola no sábado depois do almoço"

Passa a: `!uG0st0d3v3r@b0l4n0s4b@d0!` (16 caracteres)

Ou usa 4 palavras aleatórias:

> `machamba$laranja7chapa!`

## Gestor de passwords

Não precisas decorar todas. Usa um **gestor de passwords** gratuito:

- [Bitwarden](https://bitwarden.com) (recomendado, código aberto)
- [KeePass](https://keepass.info) (offline)

Só precisas de decorar **uma** password (a do gestor).

## Phishing (como roubam as tuas passwords)

Nunca cliques em links que pedem a tua password, mesmo que pareçam do:
- Banco comercial, BIM, Standard Bank, etc.
- Facebook, WhatsApp, Instagram
- Multicaixa, EMIS, etc.

**Bancos e serviços sérios nunca pedem a tua password por link.**

# Como Identificar Phishing (Golpes Online)

Phishing é quando alguém tenta enganar-te para roubar a tua senha, dados pessoais ou dinheiro.

## Sinais de alerta

### 1. O remetente é suspeito

Verifica sempre o e-mail ou número de quem enviou:

| Parece | Mas na verdade é |
|--------|-----------------|
| `bim@bim.co.mz` | `bim@bim-seguranca-mz.tk` |
| `facebook@mail.com` | `faceb00k-suporte@hotmail.com` |
| WhatsApp de número internacional | Número com +222, +44, etc. (não é Moçambique) |

### 2. Urgência ou ameaça

Mensagens tipo:
- "A sua conta será bloqueada em 24h"
- "Clique aqui para evitar multa"
- "Atualize os seus dados urgentemente"

**Isto é pressão emocional para não pensares.**

### 3. Links suspeitos

Passa o rato por cima do link (sem clicar) para ver o destino real.

Exemplo: `https://bim.co.mz.atualizacao.tk/login` não é o BIM.

### 4. Erros de português

- "Sua conta foi desativada por medidos de segurança"
- Frases estranhas, pontuação errada, português de tradução automática

### 5. Pedem dados sensíveis

Bancos, operadoras e redes sociais **nunca** pedem por e-mail ou SMS:
- Password
- PIN do Multicaixa
- Código de confirmação
- Número do BI ou NUIT

## Exemplos reais em Moçambique

- **Falso BIM/Standard Bank**: SMS ou e-mail com link para "atualizar conta"
- **Falsa Multicaixa**: mensagem que "o seu cartão foi bloqueado"
- **Falso prémio**: "Você ganhou 50.000 MZN, clique para receber"
- **WhatsApp**: número estranho a dizer que é familiar com novo número

## O que fazer

1. **Nunca cliques** no link
2. **Apaga** a mensagem
3. Se achares que pode ser real, **contacta a entidade oficial** pelo número conhecido (não pelo número da mensagem)
4. Avisa familiares e amigos para não caírem
